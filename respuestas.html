¿Qué es y que función cumple el CSS?

CSS son las abreviaciones de (cascading style sheets), CSS permite crear paginas web muy atractivas, cascading style sheets es un lenguaje informático que especifica cómo se presentan los documentos a los usuarios: cómo se diseñan, compaginan, etc.

¿Cómo se incorpora el CSS dentro del HTML?

Para vincular styles.css a index.html, añade la siguiente línea en algún lugar dentro del del documento HTML: <link rel="stylesheet" href="styles.css"<
