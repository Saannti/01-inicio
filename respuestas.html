¿Qué significa HTML y por qué se dice que es un lenguaje de marcas? Enumera 8 etiquetas HTML

HTML significa lenguaje de marcado de hipertexto se dice que es un lenguaje de marcas ya que su principal caracteristica es que sus elementos se representan en etiquetas las cuales se encierran mediante simbolos

1- <h1> 2- <p> 3- <tiltle> 4- <head> 5- <h2> 6- <h3> 5- <h4> 7- <h5> 8- <h6>

¿Cómo es la anatomía de un elemento HTML? Dar un ejemplo con la etiqueta

la anotomia de HTML contine una <p>etiqueta de apertura, una etiqueta de cierre<p>, y el contenido, ejemplo: <p>me llamo santino</p>

¿Cuál es la anatomía básica del HTML? Escribir su código y definir <html> <0head></head> <meta> <charset="utf-8"> <title></title> <body></body></h2>

la anatomia basica seria: <!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta http-equiv="X-UA-Compatible" content="IE=edge"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Document</title> </head> <body> </body> </html>
